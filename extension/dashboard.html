✏️ Naming Conventions

Define how folders and files are named when stories are saved to disk. Use the tokens below — if a token's data is not available for a story it is removed cleanly, with no dangling dashes or empty segments. Illegal Windows characters and emojis are stripped automatically. File names are truncated to a maximum of 100 characters.
