# Model weight files for @vladmandic/human
#
# These files are maintained automatically by the GitHub Actions workflow
# `.github/workflows/update-libs.yml`, which runs after every merged PR and
# on a weekly schedule.  The workflow queries the npm registry for the latest
# published version of @vladmandic/human, downloads the required files, and
# commits them to the repository — no manual action required.
#
# The currently pinned version is recorded in .human-version at the repo root.
#
# Models used:
#   blazeface.json / blazeface.bin   – lightweight face detector
#   faceres.json   / faceres.bin     – face embedding (descriptor)
